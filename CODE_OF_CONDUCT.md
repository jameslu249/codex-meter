# Code Of Conduct

Codex Meter uses the Contributor Covenant Code of Conduct, version 2.1, in spirit.

## Our Standards

Helpful behavior includes:

- being respectful and specific
- assuming good intent while staying honest about bugs
- keeping private credentials and account data out of public issues
- helping newcomers understand the project boundaries

Unacceptable behavior includes harassment, personal attacks, credential exposure, or publishing private account data.

## Enforcement

Maintainers may edit, hide, lock, or remove issues and pull requests that expose secrets or private account data.

Security reports should follow [SECURITY.md](SECURITY.md).
