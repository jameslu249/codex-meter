# Support

Codex Meter is an open-source local utility.

## Before Opening An Issue

Please check:

- You are signed in to the Codex app.
- `~/.codex/auth.json` exists.
- The app can reach `chatgpt.com`.
- You are running macOS 13 or newer.

## What To Include

Include:

- macOS version
- Codex Meter version or commit SHA
- whether the issue affects loading, menu-bar behavior, settings, or meter display
- a screenshot with private information redacted

Do not include:

- access tokens
- cookies
- `~/.codex/auth.json`
- account ids
- raw private endpoint responses

## Security

For security-sensitive reports, read [SECURITY.md](SECURITY.md).
