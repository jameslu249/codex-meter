## Narrative

1. Heavy Codex users need a glanceable way to know what is available.
2. Codex Meter lives quietly in the macOS menu bar.
3. Reset Bank shows available resets, granted dates, and expiration dates.
4. Usage meters show Codex and Codex-Spark windows.
5. Colors communicate remaining capacity.
6. Settings let users choose Circular, Bars, or Battery.
7. It is local-first and open source.

## On-Screen Copy

- Codex Meter
- Your Codex usage at a glance
- Reset Bank
- Granted and expiration dates without scrolling
- Codex and Codex-Spark grouped clearly
- Circular, Bars, or Battery meters
- Local-first. No analytics. Open source.

## Release Caption

Codex Meter is a native macOS menu-bar widget for Codex power users. It shows Reset Bank, Codex and Codex-Spark usage, weekly reset timing, and health-colored meters using your existing local Codex session.
