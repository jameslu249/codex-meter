# Codex Meter Demo Design

## Style Prompt

A quiet, premium macOS utility demo with frosted aqua glass, soft daylight, crisp system typography, and controlled motion. The video should feel like a useful developer tool, not a launch-page spectacle: calm, precise, readable, and open-source friendly.

## Colors

- `#EAF8FB` background wash
- `#C8F3F9` glass highlight
- `#2DC7DF` Codex accent
- `#43C96D` healthy usage
- `#FFB53D` warning usage
- `#20272B` primary text
- `#6B7478` secondary text

## Typography

- Primary: Apple system font stack, `-apple-system`, `BlinkMacSystemFont`, `SF Pro Display`, `Helvetica Neue`, `Arial`, sans-serif.
- Numbers use tabular figures.
- Keep demo text large enough for a GitHub release page thumbnail.

## Motion

- Soft fades and short directional slides.
- Subtle screenshot scale only; no aggressive zooms.
- Use crossfades between scenes.
- Let screenshots carry the product truth.

## What NOT To Do

- Do not use fake personal data or real private account details.
- Do not use neon-on-black, purple-blue launch gradients, or oversized marketing copy.
- Do not crop the widget so the Reset Bank or meters become unreadable.
- Do not add correction patches or fake UI overlays on top of screenshots.
- Do not imply OpenAI endorsement.
